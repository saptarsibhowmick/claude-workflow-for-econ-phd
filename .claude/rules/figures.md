# Figure Conventions

- No embedded titles — titles go in the LaTeX `\caption{}`
- Publication width: 6.5 inches (single column) or 3.25 inches (half-width)
- Viridis or colorblind-safe palette — no red-green distinctions
- Serif fonts on axis labels to match paper body text
- Full year on time axes (2015, 2016, ...) not abbreviated
- Minimal gridlines — light gray horizontal only
- PDF format for LaTeX, PNG at 300 DPI for drafts
- Every figure generated programmatically — no manual edits
- Event study plots: always show pre-treatment coefficients, confidence intervals, and reference period
